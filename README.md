## Installation Instructions
1. Clone the repository:
    ```sh
    git clone <https://github.com/WalterOlando/test-assistant-repo.git>
    ```
2. Navigate to the repository directory:
    ```sh
    cd test-assistant-repo
    ```
3. Install dependencies:
    ```sh
    npm install
    ```
